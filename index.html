تعريف Markup language

Markup language: هو نوع من اللغات
  • يتكون من عناصر (Elements)
  • من امثلته:
    1. HTML لتصميم مواقع الويب
    2. XAML لتصميم تطبيقات سطح المكتب
    3. XML وهي لغة تتعدد استخداماتها مثل حفظ البيانات
  • غالبًا ينتهي اسم الملف وامتداد الملف بML مثل HTML

تركيب العناصر في HTML

الجزء الأول

تتكون العناصر في Markup languages من ثلاث أجزاء:
  • الافتتاح: هو الجزء المطلوب لاستخدام العنصر:
    1. يبدأ بعلامة >
    2. تليها اسم العنصر
    3. ثم علامة <
    4. مثل <p>
    5. تقع في بداية العنصر

تركيب العناصر في HTML

الجزء الثاني

  • المنتصف: هو جزء ليس موجود في كل العناصر:
    1. مطلوب للعناصر التي تقوم بعرض النصوص بداخلها أو التي تحتوي على عناصر أخرى أو نصوص تقوم بمعالجتها دون عرضها.
    2. تتكون من نص أو عنصر آخر أو كلاهما.
    3. تقع في منتصف العنصر.

تركيب العناصر في HTML

الجزء الثالث

  • النهاية: هو جزء ليس موجود في كل العناصر:
    1. تبدأ بعلامة />
    2. تليها اسم العنصر
    3. ثم علامة <
    4. مثل <p/>
    5. تقع في نهاية العنصر.
    6. مهمة للعناصر التي تحتوي نصوص
    7. غالبًا غير موجودة في العناصر التي تحتوي خواص فقط
  • الملخص: أمثلة على العناصر:

    <>السلام عليكم</>

خصائص العناصر

خصائص العناصر: هي خصائص توضع للعناصر:
  • توضع بعد اسم العنصر
  • تحدد أشياء مهمة تخص العناصر
  • مثل name و id و class
  • مثال لاستخدام الخواص على عنصر:

    < =>السلام عليكم</>

  • تتكون من اسم الخاصية ويساوي وعلامتي تنصيص ويوضع بينهما الخاصية.
  • أحيانًا لا تحتاج الخاصية سوى إضافة اسمها للعنصر وبدون علامة يساوي أو علامات التنصيص مثل hidden
  • الخاصية name تدل على اسم مجموعة العنصر.
  • الخاصية id تدل على العنصر عند الاشارة اليه في اللغات الأخرى المستخدمة مع HTML.
  • الخاصية class تدل على اسماء مجموعات العنصر.
  • الخاصية hidden تقوم باخفاء العنصر من المستعرض

الجزء الأول

انشاء الملف في Visual Studio Code

يتم انشاء الملف من خلال بعض الخطوات:
  1. من الشاشة الافتتاحية تحديد فتح مجلد

    2. "Open Folder"

  2. اختيار المجلد
  3. اضغط على قائمة علامة مستعرض المجلد إن لم تظهر القائمة

    4. Explorer

  4. من الشريط الذي يظهر فيه اسم المجلد اضغط زر انشاء الملف وحدد الاسم

    5. "New File..."

    يكون اسم ملف HTML غالبًا index.html أو index.htm

الجزء الثاني

كتابة سطور HTML في الملف

  1. من قائمة المستعرض اختر اسم الملف لفتحه إن لم يتم فتحه تلقائيًا
  2. ان لم يظهر الملف حدد اسم الملف من الشريط
  3. يتم كتابة كود HTML في منطقة الكتابة:

كود HTML الأساسي

  • يوجد في لغة HTML كود أساسي:

  • يساعد الكود الأساسي في HTML على استخدام جميع المميزات.
  • يمكن كتابة هذا السطر في Visual Studio Code من صندوق الاقتراحات بكتابة علامة تعجب واحدة كما بين القوسين: (!)

    • يمكن فتح صندوق الاقتراحات إن لم يتم فتحه تلقائيًا باستخدام Ctrl + Space من لوحة المفاتيح.

السطر الأول من الكود الأساسي

  • السطر الأول من الكود الأساسي لHTML:

    <! >

  • هو سطر مهم لاستخدام آخر اصدار من HTML

    • آخر اصدار من HTML يسمى HTML5

  • يمكن كتابة هذا السطر في Visual Studio Code من صندوق الاقتراحات بكتابة ثلاث علامات تعجب كما بين القوسين: (!!!)

    • يمكن فتح صندوق الاقتراحات إن لم يتم فتحه تلقائيًا باستخدام Ctrl + Space من لوحة المفاتيح.

  • يقوم هذا السطر بتحديد كيفية عرض الصفحة

السطور الأولى من الكود الأساسي

بعد أول سطر من الكود الأساسي يوجد:
  • سطر بداية الكود ليكون بداية للكود بعد السطر الأول:

    < =>

    الخاصية lang تحدد لغة عرض الموقع و en تحدد اللغة الانجليزية

  • السطر الذي يليه هو بداية رأس الموقع الذي يضم أهم بياناته وغالبًا تعالج تلك البيانات دون عرضها:

    <>

عناصر meta ودورها في المواقع

بعد السطور الأولى من الكود الأساسي يوجد عنصران يسميان meta:
  • يحدد الأول طريقة ترميز الكلمات في الكود:

    < =>

  • يحدد الثاني طريقة عرض الصفحة من الحجم:

    < =; =>

    • قم بعمل بحث عن UTF-8 وما دوره في تصميم المواقع؟

عنوان الموقع

بعد عناصر meta يوجد عنصر عنوان الموقع:
  • هو عنصر يحدد عنوان الموقع:

    <>Document</>

  • باستخدام هذا السطر يكون عنوان الموقع هو Document
  • عند تغيير النص بداخله يتغير عنوان الصفحة
  • عند تغيير عنوان الصفحة لا يزال يمكنك استخدام باقي المزايا الخاصة باللغة دون تغيير مميزات الكود الأساسي
  • يمكن على سبيل المثال تغيير العنوان الى "صفحة البرمجة" عن طريق:

    <>صفحة البرمجة</>

    • كيف يتم القيام بتغيير عنوان صفحة الى اسم مدرستك؟

خاتمة الكود الأساسي

بعد العنوان تأتي بعض السطور خاتمةً للكود الأساسي وهي:
  • نهاية عنصر رأس الصفحة:

    </>

  • بداية عنصر جسم الصفحة وهو العنصر الذي يتم وضع العناصر فيه وغالبًا ما يتم عرض تلك العناصر:

    <>

  • يوجد مسافة بين عنصر جسم الصفحة والسطور الباقية ليتم وضع العناصر فيه.
  • نهاية عنصر جسم الصفحة:

    </>

  • نهاية العنصر المحتوي للصفحة (عنصر html)

    </>

عناصر HTML

داخل عنصر الجسم يوضع بعض العناصر مثل:
  • للنصوص:
    العناصر العناوين الفقرات
    عدد العناصر 6 1
    تكوين اسماء العناصر حرف h + رقم الحجم من 1 الى 6 يتكون من حرف p فقط
    خصائص اضافية كلما زاد الرقم قل الحجم يحتوي فقرات - لا تستخدم كعناوين - نصوصها أصغر من العناوين
    أمثلة h1 - h2 - h6 - h7 p
    طريقة الاستخدام

    <>النص يوضع هنا</>

    يجب استبدال "element" باسم العنصر و"النص يوضع هنا" بالنص الفعلي.
