تعريف HTML

HTML: هي لغة ترميز
  1. تستخدم في تصميم مواقع الويب
  2. سهلة الاستخدام
  3. هي لغة Markup language
  4. يمكن استخدامها بشكل أساسي مع JavaScript و CSS و PHP
  5. تسمي Hypertext Markup language

تعريف Markup language

Markup language: هو نوع من اللغات
  • يتكون من عناصر (Elements)
  • من امثلته:
    1. HTML لتصميم مواقع الويب
    2. XAML لتصميم تطبيقات سطح المكتب
    3. XML وهي لغة تتعدد استخداماتها مثل حفظ البيانات
  • غالبًا ينتهي اسم الملف وامتداد الملف بML مثل HTML

تركيب العناصر في HTML

الجزء الأول

تتكون العناصر في Markup languages من ثلاث أجزاء:
  • الافتتاح: هو الجزء المطلوب لاستخدام العنصر:
    1. يبدأ بعلامة >
    2. تليها اسم العنصر
    3. ثم علامة <
    4. مثل <p>
    5. تقع في بداية العنصر

تركيب العناصر في HTML

الجزء الثاني

  • المنتصف: هو جزء ليس موجود في كل العناصر:
    1. مطلوب للعناصر التي تقوم بعرض النصوص بداخلها أو التي تحتوي على عناصر أخرى أو نصوص تقوم بمعالجتها دون عرضها.
    2. تتكون من نص أو عنصر آخر أو كلاهما.
    3. تقع في منتصف العنصر.

تركيب العناصر في HTML

الجزء الثالث

  • النهاية: هو جزء ليس موجود في كل العناصر:
    1. تبدأ بعلامة />
    2. تليها اسم العنصر
    3. ثم علامة <
    4. مثل <p/>
    5. تقع في نهاية العنصر.
  • الملخص: أمثلة على العناصر:

    <>السلام عليكم</>

خصائص العناصر

خصائص العناصر: هي خصائص توضع للعناصر:
  • توضع بعد اسم العنصر
  • تحدد أشياء مهمة تخص العناصر
  • مثل name و id و class
  • مثال لاستخدام الخواص على عنصر:

    < =>السلام عليكم</>

  • الخاصية name تدل على اسم مجموعة العنصر.
  • الخاصية id تدل على العنصر عند الاشارة اليه في اللغات الأخرى المستخدمة مع HTML.
  • الخاصية class تدل على اسماء مجموعات العنصر.
  • تتكون من اسم الخاصية ويساوي وعلامتي تنصيص.
